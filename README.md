# LangChain-agent


# Setup
Steps:
    1. Install uv and run uv init to bootstrap a python project.
    2. Add langchain with - uv add langchain
    3. Uv also creates a venv for the project
    4. Install integration packages for LLM providers like openai etc.
    5. uv add python-dotenv 
    6. Format code - uv add black isort
